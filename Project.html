    2. Attendance Management System using Face Recognition Feature
    (Jan'18 - May'18)

    a) Objective:- To mark attendance of a particular class without teacher's involvement.
    b) Description:-
    Firstly, a camera is setup to scan the face and eye retina.
    Secondly, the scanned image is trained and converted to greyscale.
    Lastly, the trained image is matched with images in class database and if the image is present in the database , the present is marked at that student's name otherwise it is marked as a new student and required details are taken to store in the database.
    c)Languages used:- Python as front end and open cv and haarcascade files for scanning purposes and structured query language as backend.
